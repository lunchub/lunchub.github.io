Categories: [ CoffeeScript, Vim ]

Environment

Issues

VimでSyntasticを導入しているものの、CoffeeScriptの保存時に シンタックスチェックが走らない。

Solution

CoffeeScriptとCoffeeLintをインストールしてください(ﾉ´∀｀*)

バージョンは1.6.2以上が必要です。

そしてパスを通す。NODE_PATHも一応通しておいたほうがいいらしい。
