“couldn’t find file ‘jquery-ui’”

問題は、

  • activeadminがjQueryとjQuery UIを使用する
  • jquery-rails, 3.0.0からjQuery UIが切り離された
  • activeadminのapplication.js内でjQuery UIの呼び方が間違っている

activeadminがjQueryとjQuery UIを使用する

jquery-rails, 3.0.0からjQuery UIが切り離された

要するにjQuery UIが切り離される前のバージョンにしろってことなんだけど、
