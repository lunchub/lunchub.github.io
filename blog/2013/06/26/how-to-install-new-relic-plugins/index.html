How to Install New Relic Plugins

Jun 26th, 2013

Tweet

newrelic plugin

Python 2.6.6 pipが必要(Python版gem?) pip requires setuptools.

solution

wget https://bitbucket.org/pypa/setuptools/raw/0.7.4/ez_setup.py -O – | python

curl -O https://raw.github.com/pypa/pip/master/contrib/get-pip.py sudo python get-pip.py

newrelicのプラグインエージェントのインストール

pipコマンドでnewrelic-plugin-agentをインストールする

pip install newrelic-plugin-agent

設定ファイルを/etc/newrelicに配置する

cp /opt/newrelic_plugin_agent/etc/newrelic/newrelic_plugin_agent.cfg /etc/newrelic/newrelic_plugin_agent.cfg

設定ファイルにライセンスキーを追記する

vim /etc/newrelic/newrelic_plugin_agent.cfg

%YAML 1.2

Application: license_key: XXXXXXXXXXXXXXXXXXXXXXXXXXXXXX poll_interval: 60

ユーザ権限の確認

これらのディレクトリが書き込み可能か、所有者であることを確認する [root@v157-7-143-69 newrelic-pluguins]# ls -la /var/run/newrelic/ total 12 drwxr-xr-x 2 newrelic newrelic 4096 Jun 26 12:32 . drwxr-xr-x 18 root root 4096 Jun 26 12:11 .. -rw-r—r— 1 newrelic newrelic 5 Jun 26 12:12 nrsysmond.pid -rw-r—r— 1 root root 0 Jun 26 12:32 safe_to_delete.txt

[root@v157-7-143-69 newrelic-pluguins]# ls -la /var/log/newrelic/ total 20 drwxr-xr-x 2 newrelic newrelic 4096 Jun 26 12:32 . drwxr-xr-x 11 root root 4096 Jun 26 12:11 .. -rw———– 1 root root 270 Jun 26 12:11 newrelic-daemon.log -rw-r—r— 1 newrelic newrelic 367 Jun 26 12:13 nrsysmond.log -rw———– 1 apache apache 270 Jun 26 12:11 php_agent.log -rw-r—r— 1 root root 0 Jun 26 12:32 safe_to_delete.txt

newrelic-plugin-agentを起動する

newrelic_plugin_agent -c PATH-TO-CONF-FILE [-f]

*-fオプションはdeamonではなくforegroundで実行したい時に使う。
