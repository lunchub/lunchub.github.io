OctopressにDISQUSのコメントウィジェットを設置する方法

Issues

OctopressのサイドバーにDISQUSの最近のコメント一覧を表示させたい。

Solution

以下の手順で設置できたのでメモ。

サイドバー表示用HTMLの作成

_config.ymlの編集

オプション

基本的に上記手順で表示できるが、いくつかオプションがある模様。
JavaScriptの呼び出しクエリに適宜付加してください。

  • hide_avatars :
    • アバターを表示させないか（1だと非表示、0だと表示）
    • 自分のコメントは1にすると見えなくなるっぽい？
  • num_items :
    • コメントを何件表示させるか
  • excerpt_length :
    • コメントの最大表示文字数
